## 📖 Overview

**Persian Loghoz** is the first comprehensive dataset of Persian figurative language, specifically focused on **irony, metaphor, and multi-layered semantic expressions**. This dataset bridges a critical gap in Natural Language Processing (NLP) resources for Persian, a language rich in literary nuance, poetic ambiguity, and deep cultural connotations.

The name **"Loghoz"** (لغز) is derived from the Persian word for *riddle* or *puzzle*, reflecting the challenging nature of figurative language understanding.

### 🎯 Key Features
- ✅ **First** Persian figurative language dataset
- ✅ **5,000+** annotated entries
- ✅ **Three complexity levels**: simple, multi-layered, complex
- ✅ **Multiple sources**: Classical and modern Persian literature
- ✅ **Rich annotations**: Source, type, complexity, context

---

## 📊 Dataset Composition

| Category | Description | Source |
|----------|-------------|--------|
| **Simple** | Common idiomatic phrases and everyday metaphors | Dehkhoda Dictionary, Amid Dictionary |
| **Multi-layered** | Expressions requiring contextual interpretation | Classical texts, proverbs |
| **Complex** | Deep poetic and rhetorical figures | Hafez Divān, Golestān of Sa'di |

### Dataset Statistics
- **Total Entries**: 5,000+ annotated expressions
- **Languages**: Persian (Farsi) with English translations
- **Format**: CSV / JSON
- **Annotation Fields**: ID, expression, meaning, literal translation, source, complexity, figurative type, context
- **Complexity Distribution**: 30% Simple / 40% Multi-layered / 30% Complex
- **Figurative Types**: Irony, Metaphor, Symbolism, Simile, Proverb, Idiom
